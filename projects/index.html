TagSafe / Tag validator prototype

Compress third-party tag QA into one screen.

Internal prototype for validating third-party display tags with bulk upload, previews, and spec checks.

Third-party tag QA often means bouncing between validator output, dev tools, tagsheets, and vendor documentation. TagSafe is a prototype for putting the most common checks in one place.

Problem

Core checks lived across multiple tools and a vendor platform. There was no lightweight way to scan a full tagsheet, compare vendors, and catch obvious setup issues quickly.

Solution

Built a browser page where ad ops can paste a tag or upload a tagsheet, auto-detect vendors, render previews, and surface load time, requests, size, dimensions, vendor, placement ID, and landing-page response.

Impact

Shipped a working proof-of-concept for standardizing repeat QA checks and making tag review easier to explain across ops and account teams.

AdBuilder / Layout engine prototype

Map one asset set across multiple display sizes.

Internal prototype that turns a single set of assets into PNG creatives across standard display formats.

Many local advertisers bring photos and copy but do not have design resources. Simple campaigns can stall while every size is built by hand.

Problem

Producing standard units from one concept meant recreating layouts in a design tool one size at a time.

Solution

Built a browser layout engine where users paste headline, description, CTA, and upload images once. It renders the size set, supports per-size copy tweaks, and exports PNGs.

Impact

Demonstrates how non-designers could generate full creative sets without art software, reducing friction for simple campaigns.

Vast2MP4 / Video utility

See what is inside a VAST tag at a glance.

Pulls MP4 files and trackers from VAST XML for export and validation.

Some custom video placements cannot accept VAST tags; they need a standalone MP4 file and the trackers rebuilt separately.

Problem

When an agency sends only a VAST tag, ops may still need to recover the media file and verify the trackers before a custom placement can launch.

Solution

Built a browser tool where anyone can paste a VAST tag, unwrap the XML, inspect metadata, download the MP4 asset, and export impression/click trackers.

Impact

Turns a one-off escalation into a repeatable debug path for video QA and custom content setups when the delivered asset format does not match the placement.

PageID Scraper / CMS helper

Bridge CMS URLs and ad server targeting.

Operational tool that turns article URLs into page IDs for faster brand-safety blocking.

Brand-safety reports arrive as long lists of article URLs and hit counts, but GAM exclusions need internal page IDs.

Problem

Excluding unsafe articles meant opening each URL, finding the pageID in source or delivery diagnostics, recording it, and manually comma-separating the final list.

Solution

Built a tool that ingests IAS exports, normalizes and dedupes URLs, rolls up hit counts, applies an impact threshold, then calls a scraping worker to return only the page IDs worth blocking.

Impact

Turns a manual lookup flow into a threshold-driven export that can cover most of the reported unsafe impressions with a much smaller, GAM-ready list.

BlockList / Brand safety

Turn messy URLs into usable blocklists.

Operational tool that builds negative keyword lists from unsafe article URLs.

Sometimes the right fix is not blocking individual pages. It is extracting the themes behind unsafe URLs and turning those into keyword exclusions.

Problem

Ad ops had to scan headline-like URL slugs, decide which terms mattered, ignore noisy words, and assemble comma-separated exclusions by hand.

Solution

Built a tool that ingests IAS URL exports, extracts hyphenated terms, dedupes them, shows frequency and impression impact, and lets users click terms into a custom export list.

Impact

Converts subjective spreadsheet review into a faster, more consistent workflow for translating client brand-suitability needs into actionable blocking terms.

Collate / Spreadsheet helper

Convert a column into a clean list in one click.

Drag-and-drop keyword cleaner that outputs ready-to-use targeting strings.

Brand-safety keyword lists can arrive as tens of thousands of spreadsheet rows. The task is simple, but the manual cleanup is repetitive and easy to get wrong.

Problem

Turning 30K-60K rows into a usable negative keyword list meant repeated Excel formulas, duplicate checks, formatting cleanup, and manual copy/paste.

Solution

Built a drag-and-drop spreadsheet helper that detects keyword columns, removes duplicates, strips punctuation, and outputs a clean comma-separated string or text file.

Impact

Turns a multi-step spreadsheet task into a drop-and-copy flow that can process tens of thousands of terms in seconds.

Outline / Image border tool

Make white-edge creatives compliant in seconds.

Used by account managers, ad ops, and a trafficking vendor at the Post.

Many client assets arrive with white edges. Policy requires a visible border so ads are not confused with surrounding page content.

Problem

Adding borders meant opening each creative in Photoshop, applying a stroke, exporting, and renaming files one by one across every concept and size.

Solution

Built a browser tool where users can upload one or many creatives, preview the border in real time, and download renamed files with the border marker appended.

Impact

Turns a 15-20 minute batch compliance task into an upload-preview-export workflow that usually takes under a minute.
