TagSafe / Tag validator prototype

Compress third-party tag QA into one screen.

Internal prototype for validating third-party display tags with bulk upload, previews, and spec checks.

Our QA teams rely on paid validators plus dev tools to review third-party tags. That works, but it is license-bound and slow when you just want to sanity-check a tagsheet.

Problem

Core checks lived across multiple tools and a vendor platform. There was no lightweight way to scan a full tagsheet quickly across vendors.

Solution

Built a browser page where ad ops can paste a tag or upload a tagsheet, auto-detect vendors, render previews, and surface load time, requests, size, dimensions, vendor, placement ID, and landing-page response.

Impact

Shipped a working proof-of-concept that recreates our most-used AdValidation checks in-house and shows a path to reducing validator licensing costs.

AdBuilder / Layout engine prototype

Map one asset set across multiple display sizes.

Internal prototype that turns a single set of assets into PNG creatives across standard display formats.

Many local advertisers bring photos and copy but do not have design resources. Simple campaigns can stall while every size is built by hand.

Problem

Producing standard units from one concept meant recreating layouts in a design tool one size at a time.

Solution

Built a browser layout engine where users paste headline, description, CTA, and upload images once. It renders the size set, supports per-size copy tweaks, and exports PNGs.

Impact

Demonstrates how non-designers could generate full creative sets without art software, reducing friction for simple campaigns.

Vast2MP4 / Video utility

See what is inside a VAST tag at a glance.

Pulls MP4 files and trackers from VAST XML for export and validation.

Some custom video placements cannot accept VAST tags; they need standalone MP4 files and 1x1 trackers.

Problem

Agencies often send only a VAST tag, leaving no time to request new files. Extracting assets manually from XML or network tools is slow and brittle.

Solution

Built a browser tool where anyone can paste a VAST tag, preview metadata, download the MP4 asset, and export impression/click trackers.

Impact

Used by ad ops and account teams to keep campaigns live without going back to clients, including formats that normally cannot accept VAST.

PageID Scraper / CMS helper

Bridge CMS URLs and ad server targeting.

Operational tool that turns article URLs into page IDs for faster brand-safety blocking.

Brand-safety reports arrive as long lists of article URLs and hits, but the ad server targets on internal page IDs.

Problem

Excluding unsafe articles meant opening every URL, finding the pageID in markup, and copy/pasting it into a blocklist.

Solution

Built a tool that ingests URLs, normalizes and dedupes them, rolls up impression counts, applies a threshold, and returns pageIDs for high-impact pages.

Impact

Turns a multi-hour CMS task into a paste-and-download workflow used regularly by ad ops.

BlockList / Brand safety

Turn messy URLs into usable blocklists.

Operational tool that builds negative keyword lists from unsafe article URLs.

IAS reports arrive as hundreds of URLs and impression counts, but GAM blocklists rely on keywords and key-values.

Problem

Ad ops had to review each URL, decide what it meant, extract terms by hand, and build negative keyword lists in spreadsheets.

Solution

Built a tool that ingests URLs, extracts hyphenated terms, dedupes them, surfaces frequency and impression impact, and lets users build a custom blocklist interactively.

Impact

Converts a subjective spreadsheet process into a consistent 5-10 minute workflow with export-ready GAM exclusion terms.

Collate / Spreadsheet helper

Convert a column into a clean list in one click.

Drag-and-drop keyword cleaner that outputs ready-to-use targeting strings.

Brand-safety keyword lists often arrive as long columns from Excel exports. Cleaning them manually takes time and introduces errors.

Problem

Turning thousands of spreadsheet rows into a usable negative keyword list meant hunting duplicates and checking stray formatting issues by hand.

Solution

Collate auto-detects keyword columns, removes duplicates, strips punctuation, flags high-risk stopwords, sorts terms, and outputs a clean string or text file.

Impact

Turns a 10-15 minute Excel chore into a five-second drop-and-copy step.

Outline / Image border tool

Make white-edge creatives compliant in seconds.

Used by account managers, ad ops, and a trafficking vendor at the Post.

Many client assets arrive with white edges. Policy requires a visible border so ads are not confused with article imagery.

Problem

Adding borders meant re-requesting assets or opening each creative in Photoshop or Canva. Several concepts across six or more sizes could become a 10-20 minute chore.

Solution

Built a browser tool where anyone can upload standard display assets, apply a policy-compliant border, and download renamed files.

Impact

Turned a license-dependent, one-minute-per-asset step into a roughly 15-second process for the full creative set.
